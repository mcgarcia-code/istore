# iStore — Mini E‑commerce (React + Tailwind + Framer Motion)

Tienda ficticia de iPhones para portfolio. Incluye catálogo con filtros, carrito con persistencia en `localStorage` y checkout de ejemplo.

## Requisitos
- Node.js 18+

## Instalación
```bash
npm install
npm run dev
```

## Build y preview
```bash
npm run build
npm run preview
```

## Stack
- Vite + React
- TailwindCSS
- Framer Motion
- Lucide React (iconos)
